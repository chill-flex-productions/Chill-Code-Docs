# Chill Code Companion Privacy Notice

Last updated: 2026-05-18

This notice describes the Chill Code Companion plugin for JetBrains IDEs. The plugin is designed to connect a user-started local IDE session with the Chill Code mobile app over the user's chosen local network connection.

## Data The Plugin Processes

Chill Code Companion may process local IDE session data that the user chooses to expose through a paired remote session. This can include project names and paths, source file content shown or edited through the remote session, terminal input and output, run output, Git status and diff metadata, pairing material, and AI chat prompts and responses when IDE AI features are used.

## Local Storage

The plugin may store short-lived remembered-pair records in the IDE's local Password Safe. It may also write diagnostic entries to IDE logs. The plugin does not require a developer-operated backend for the local IDE-to-mobile companion session.

## Sharing And Third Parties

Project and session data is exchanged with the paired Chill Code mobile app to operate the user-started companion session. When IDE AI features are used, AI prompts and responses are handled by the AI integration configured in the user's IDE. The plugin does not choose that AI provider directly.

Chill Code Companion does not sell user information or share it for advertising, marketing, analytics, or unrelated third-party use.

## User Controls

Users choose when to start a companion session, which mobile device to pair, and when to stop the session. Users can remove remembered-pair material through IDE settings, by clearing the IDE Password Safe entry, or by uninstalling the plugin.
